# takymt’s Homebrew Tap

Homebrew tap for shipping my CLI tools.

## Install

```bash
brew tap takymt/tap
```

## Install Packages

```bash
# formula
brew install takymt/tap/<name>

# cask
brew install --cask takymt/tap/<name>
```

## Packages

### Formulae

- [`cflcli`](https://github.com/takymt/cflcli) — Confluence CLI for managing Pages with local Markdown files.

### Casks

Currently no casks 🥹

## Update / Uninstall

```bash
brew update
brew upgrade

brew uninstall <formula>
brew uninstall --cask <cask>
```
